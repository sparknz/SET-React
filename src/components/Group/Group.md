Used to group things together

```
import Button from '../Button';

<Group variant="row" desktopSpacing="default" tabletSpacing="small">
    <Button>1</Button>
    <Button>2</Button>
    <Button>3</Button>
</Group>
```

```
import Button from '../Button';

<Group variant="column" desktopSpacing="xxl" tabletSpacing="xxs">
    <Button>1</Button>
    <Button>2</Button>
    <Button>3</Button>
</Group>
```
